გვერდის ბანერები

გაფრთხილება!

აუცილებელია ფოტოების გაფართოება იყოს jpg

თუ არებული ფოტოები არ აკმაყოფილებს შესაბამის ფორმატს ნახეთ კონვერტერები

ბანერების დასამატებლად/შესაცვლელად უნდა გაიაროთ შემდეგი ნაბიჯები:

ფოტოს ატვირთვა ბანერისთვის​

ფოტოების ასატვირთად უნდა შეხვიდეთ Admin panel ში, Website -> images

დააჭირეთ upload ღილაკს, რის შემდეგაც გაიხსნება ფოტოების ასატვირთი სექცია.

Upload type ში ვირჩევთ Other ტიპს, შემდგომ ვყრით ფოტოებს ინფუთში და ვაწვებით Upload ღილაკს.

ბანერის დამატება / წაშლა / ჩანაცვლება​

შევდივართ Website -> pages სექციაში.

ვირჩევთ შესაბამის ფეიჯს, მაგალითად home-women, ვნიშნავთ და ვაჭერთ Edit ღილაკს.

შევდივართ EDIT BLOCKS -> EDIT0 (მთავარი ბანერის შესაცვლელად, სხვა შემთხვევაში უნდა მოძებნოთ შესაბამისი ღილაკი).

შემდგომ EDIT CONTENT სადაც გვხვდება EDIT EN და EDIT KA ღილაკები.

გაფრთხილება!

თუ კონტენტს არ შევცვლით ორივე ენისთვის, ცვლილებები აისახება საიტის მხოლოდ ქართულ/ინგლისურ ვერსიაზე

შევდივართ EDIT KA -> EDIT ITEMS სადაც ვხედავთ არსებულ ბანერებს.

პირობითად შევდივართ EDIT0 სექციაში და აქ ვცვლით ბანერის კონტენტს (ფოტოს, ტექსტს, ლინკს (ლინკი მოცემულია EDIT BUTTON ში))

რაც შეეხება უშუალოდ ბანერის ლინკს (რომელიც Bg-image და Bg-image-mobile ში ისმება), მის სანახავად უნდა შევიდეთ Website -> images ში და საძიებო ველში ჩავწეროთ შესაბამისი სახელი, შემდგომ კი დავაკოპიროთ Image Link სექციაში მოცემული მნიშვნელობა, რომელსაც შემდგომ ჩავსვამთ შესაბამისი ბანერის ველებში.

ცვლილებებს შევინახავთ SAVE / SAVE & CLOSE ღილაკით და საიტზე შევამოწმებთ ასახულ ცვლილებებს.
